Tipos de link

Link externo

  • Apontam para outro site e não para o nosso
  • EX: Você pode acessar o pdf dessa aula no repositorio do professor no github

Muito obrigado por assistir! Essa aula está nesse link

Link interno

  • Apontam para outro site de seu servidor
    • EX: Essa página que você está é a página principal. Mas você pode ir para outra página clicando aqui

Link para download

Devemos ponhar um link em pdf para ser efetuado o download, inclusive, você pode compactar o arquivo para arquivo zip e diminuir o tamanho dele
