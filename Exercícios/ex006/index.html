Principais formatações

Negrito / destaque

Nessa frase temos um termo em negrito usando a tag < b > (que não é semântica)

Nessa frase temos um termo em negrito usando < strong > (que é semântica)

Itálico / ênfase

Nesta frase temos um termo em ítalico usando a tag < i > (não semântica)

Nesta frase temos um termo em ítalico usando < em > (semântica)

Texto marcado

Podemos criar também um texto marcado com a tag < mark >

Podemos mudar a cor do seu texto marcado usando a tag mark da seguinte maneira < mark style='background-color: cor'> texto ; < /mark >

Para configurar a cor de todos os textos marcados do seu código devemos abrir um style dentro de head com um comando background-color dentro dele.

Texto grande e texto pequeno

Estamod escrevendo um texto grande e um texto pequeno. O texto grande já é considerado obsoleto pois devemos alterar isso usando as CSS. no entanto os textos pequenos ainda são usados em letras pequenas de contrato por exemplo

Para deixar um texto grande devemos usar a tag < big > já para deixar o texto pequeno devemos usar a tag < small >

Texto deletado

Podemos mostrar que um texto foi excluído. Para mostrar que ele deve ser lido mas não considerado. Usamos a tag < del >

Texto inserido

Podemos marcar um texto inserido para dar um ênfase e mostrar que ele foi adicionado depois. Usamos a tag < x >

Texto subescrito

Para inserir coisas do tipo x20 (elevado). Usamos a tag < sup >

Texto subscrito

Para inserir coisas do tipo H2O (2 abaixado). Usamos a tag < sub >
