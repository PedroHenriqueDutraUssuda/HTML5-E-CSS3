Código fonte

Ajusta o espaçamento entre as strings com o comando < code >

Ex: o espaçamento do t é menor que o do g, mas no exemplo o comando do código fonte deixa os dois com o mesmo espaçamento document.getElementByld('teste')

Outro comando do código fonte é o < pre >, ele é usado para espaçamento e quebra de linha em uma string, já que o seu código deixa uma linha em frente a outra se não usar a tag BR

        Ex:
        num = int(input('Digite um número: '))
        if num % 2 == 0:
            print('Seu número é par!')
        else:
            print('Seu número é ímpar!')
        print('Obrigado! volte sempre!')

Citações

Para evitar usar as aspas em citações, já que elas possuem um valor semântico, utilizamos a tag < q >, como no exemplo:
A existência precede a essência _ Sartre

Para citações em bloco utilize o comando < blockquote >, nele você pode utilizar a tag < cite="link do site" > isso ajuda na notoriedade do seu site!

Abreviações

Para usar abreviações utilize a tag < abbr >, e coloque o significado para seu usuário passar o mouse em cima e saber o que significa!
